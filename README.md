## Analysed E-commerce Sales Data Using Power BI

### Project Overview:
This project focuses on analyzing e-commerce sales data using Microsoft Power BI to uncover actionable insights that can drive business growth. The goal is to understand key metrics such as sales performance, profit trends, customer behavior, and product-wise revenue distribution through an interactive and visually rich Power BI dashboard.

By performing data cleaning, transformation, and visualization, this project provides business intelligence insights that can help decision-makers optimize marketing strategies, inventory, and regional sales efforts.

### Tech Stack:
Data Visualization Tool: Power BI

Data Cleaning & Transformation: Power Query (Power BI)

Data Source: CSV / Excel dataset (E-commerce sales data)

Languages: DAX (for calculated columns & measures)

### Key Features:
Interactive Power BI Dashboard with multiple views

KPI cards showing key performance metrics (Sales, Profit, Quantity, etc.)

Dynamic filters and slicers for Region, Category, Sub-Category, and Time

Trend analysis for monthly/quarterly/yearly sales

Top-performing and underperforming products visualization

Regional profit and customer segmentation insights

Data-driven business recommendations

### Dataset Description:
The dataset contains detailed e-commerce transactional data with the following fields:
Order ID – Unique identifier for each order

Order Date – Date of purchase

Ship Mode – Shipping type (Standard, Express, etc.)

Customer Name / Segment – Customer details and segmentation

Region – Geographic region of sale

Category & Sub-Category – Product classification

Sales – Revenue generated

Profit – Net profit on each sale

Quantity – Number of items sold

### Dashboard Insights:
Some of the key insights derived from the dashboard include:
Highest sales generated from Technology and Office Supplies categories.

Certain regions outperform others in both profit and sales volume.

Seasonal trends highlight peak months for sales performance.

Profit margins vary significantly between product sub-categories.

Business recommendations can be made based on underperforming regions and product lines.

### Learnings:
Data cleaning & transformation using Power Query Editor

Creating calculated measures using DAX

Designing visually appealing dashboards in Power BI

Gaining actionable insights from business data

### Conclusion:
This project showcases the power of data visualization and business intelligence in understanding e-commerce trends. With Power BI, raw sales data was transformed into meaningful insights that can help businesses make informed, data-driven decisions.
